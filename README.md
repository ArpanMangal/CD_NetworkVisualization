# Confluent Drawings for Network Visualization

This is an implementation of the paper "Towards Unambiguous Edge Bundling: Investigating
Confluent Drawings for Network Visualization".

### Visit https://confluent-graphs.herokuapp.com/

### To run locally
```
    $ npm install
    $ npm start
```
Visit http://localhost:3000/
